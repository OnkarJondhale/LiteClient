**LiteClient** is a lightweight, high-performance API testing tool that runs directly in your browser as a Chrome Extension. It is designed for developers who need a fast, privacy-focused alternative to heavy desktop applications like Postman or Insomnia.

### Why LiteClient?
- **CORS-Free Testing:** By running as an extension with `host_permissions`, LiteClient bypasses browser security restrictions (CORS) automatically, allowing you to test any API.
- **Privacy First:** Your data stays on your machine. No cloud syncing of your sensitive API keys or request history.
- **Minimalist Design:** Built with Vanilla JS and Tailwind CSS for a clean, distraction-free interface.
- **New Tab Experience:** Opens in a full browser tab for maximum screen real estate.

## ðŸ›  How to Install

Since LiteClient is currently in developer mode, follow these steps to install it in your Chrome browser:

1. **Download the Code:**
   - Click the green **Code** button at the top of this page.
   - Select **Download ZIP** and extract the files to a folder on your computer.

2. **Open Extensions Page:**
   - Open Chrome and type `chrome://extensions/` in the address bar.

3. **Enable Developer Mode:**
   - Toggle the **Developer mode** switch in the top right corner to **ON**.

4. **Load the Extension:**
   - Click the **Load unpacked** button in the top left.
   - Select the `LiteClient` folder (the one containing `manifest.json`).

5. **Pin for Quick Access:**
   - Click the **Puzzle Piece** icon in your Chrome toolbar.
   - Click the **Pin** icon next to **LiteClient** to keep it visible.
